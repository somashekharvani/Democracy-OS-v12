# 🚀 **QUICK START GUIDE: v2.0 PRO EDITION**

## **1. Setup (2 Minutes)**
```bash
npm install
npm run dev
```

## **2. The "Winner" Demo Sequence**

### **Step 1: The "Wow" Factor (Visuals)**
Show the **Security Hub** first. Point out the live metrics, the 12,000+ nodes, and the 99.999% uptime. This establishes immediate technical credibility.

### **Step 2: The "Inclusion" Factor (Voice AI)**
1. Change language to **Kannada** or **Hindi**.
2. Click the Floating AI Assistant.
3. Grant microphone permissions.
4. Speak naturally. The AI will respond in the selected language.

### **Step 3: The "Security" Factor (Face ID)**
1. Go to the **Voting Center**.
2. Click "Initialize Biometric Scan".
3. Point your camera at your face.
4. Show how the system "finds" you and unlocks the secure ballot.

### **Step 4: The "Transparency" Factor (Blockchain)**
1. Cast a vote.
2. Show the transaction hash (e.g., `0x4a2b...`).
3. Explain that this is now immutable on the blockchain ledger.

## **⚠️ Requirements for Demo**
- **Webcam**: Needed for the "Face ID" biometric simulation.
- **Microphone**: Needed for the "Voice AI" demonstration.
- **Chrome/Edge**: Best browsers for the Speech Recognition API.
